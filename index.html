Advanced Technical Features

Location-Based Task System

  • Interactive map with real-time location tracking
  • Pin placement with directional indicators (cones showing camera angle)
  • Proximity-based camera activation (15-meter radius)
  • Path drawing for video tasks with turn-by-turn instructions

Camera & Media System

  • Integrated camera for photo/video capture
  • Video path following with navigation instructions
  • Media upload to Firebase Storage
  • Photo library integration for saving completed work

Task Management

  • Real-time task status tracking (Available → Assigned → In Progress → Submitted → Approved/Rejected)
  • Task history and completion tracking
  • Second chance workflow for rejected submissions
  • Task cancellation and refund handling

Technical Architecture

  • SwiftUI-based iOS app with MapKit integration
  • Firebase Firestore for real-time database
  • Firebase Cloud Functions for serverless backend
  • Stripe integration with external payment flow (App Store compliant)
